# **App Name**: HelloChoco Display

## Core Features:

- Display 'Hello' Text: Render the 'Hello' text on the page.
- Display 'Choco Smiley' Text: Render the 'Choco Smiley' text on the page.
- Centered Text Alignment: Ensure both text elements are horizontally and vertically centered on the page.
- Minimalist Layout: Maintain an uncluttered, empty page layout with focus solely on the text elements.

## Style Guidelines:

- Background color: Deep purple (#5D2B79), providing a rich and striking backdrop as specified by the user.
- Accent color: Bright golden yellow (#FFD139), specifically used for 'Choco Smiley' text to ensure it stands out, as requested by the user.
- Text color for 'Hello': Pure white (#FFFFFF), offering high contrast against the dark background.
- Headline and body font: 'Inter' (sans-serif) for all text, as explicitly requested by the user, offering a clean, modern, and highly legible appearance.
- All content should be centrally aligned, creating a focal point for the displayed text.
